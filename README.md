# News-App

Project by: **Ibrahim Sefer seferibrahim2@gmail.com** <br>
Stack used: **ASP.NET Core 5 & Angular 11** <br>

Roles: **Admin, User**
Admin: **FILTER, CREATE, READ, UPDATE, DELETE (CRUD)** <br>
User/Guest: **READ** <br>

## Getting started

``npm i`` <br>

### First terminal

``cd api`` <br>

``dotnet run``

### Second terminal

``cd client`` <br>

``ng serve``

Then go to browser link http://localhost:4200
